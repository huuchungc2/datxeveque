# 3. Rule chung

- UI tiếng Việt 100%.
- Không dùng label English như Booking/Trip/Driver.
- Button tối thiểu 48px trên mobile.
- Input tối thiểu 48px.
- Card bo tròn.
- Desktop dùng table.
- Mobile dùng card.
- Admin ưu tiên thao tác nhanh.
- Tài xế ưu tiên mobile.

```txt
new -> Mới tạo
```
