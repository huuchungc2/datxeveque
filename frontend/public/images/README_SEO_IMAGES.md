# SEO images - Đặt Xe Về Quê

Bộ ảnh này được đặt trong `frontend/public/images/` để phục vụ layout trang Home và SEO social preview.

Quy tắc đã áp dụng:

- Tên file chứa từ khóa tiếng Việt không dấu: `dat-xe-ve-que`, `sai-gon`, `duc-linh`, `tanh-linh`.
- Dùng `.webp` để nhẹ, có `.jpg` fallback cho ảnh tuyến chính.
- Ảnh hero được preload trong `index.html` và `HomePage.tsx`.
- Các ảnh tuyến dùng thẻ `<img>` có `alt` mô tả rõ tuyến xe.
- Ảnh OG: `/images/og-dat-xe-ve-que-sai-gon-duc-linh.webp`.

Danh sách ảnh chính:

- `hero-dat-xe-ve-que-sai-gon-duc-linh.webp`
- `og-dat-xe-ve-que-sai-gon-duc-linh.webp`
- `xe-sai-gon-di-duc-linh.webp`
- `xe-sai-gon-di-tanh-linh.webp`
- `xe-duc-linh-di-sai-gon.webp`
- `xe-lagi-di-sai-gon.webp`
